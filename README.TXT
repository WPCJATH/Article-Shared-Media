-Article-Shared-Media
The Media System is a platform for publishing and reading articles.
Like Weibo, the user may sign up a account and post any articles he or she wants, then the other users can view his or articles.
The user also can quote article written by someone else or the user itself.
The user can add or delete friends, check the main page of friends.
Article uploaded on the platform can be deleted.
The system will evaluate the KOL(Influential users) peoeple base on the number and influence of thier articles.

This system is written by python and store data in .txt file.
Details are in project description and the report.
